# RedisClusterWitness

watch redis cluster and failover when needed.

## Installation

how to install in linux:
```
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

how to install in windows:

```
python -m venv venv
.\venv\Scripts\activate.bat
pip install -r requirements.txt
```

## Usage

```
python main.py
```
